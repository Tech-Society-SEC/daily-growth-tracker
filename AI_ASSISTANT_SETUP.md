## 🐛 Troubleshooting:

### **If AI doesn't respond:**

1. **Check API Key:**
   - Open `.env` file
   - Verify key is correct
   - No spaces or quotes

2. **Restart Server:**
   ```bash
   # Stop: Ctrl+C
   npm run dev
   ```

3. **Check Console:**
   - Press F12
   - Look for errors
   - Check Network tab

4. **Verify API Key:**
   - Go to: https://makersuite.google.com/app/apikey
   - Check if key is active
   - Generate new key if needed
